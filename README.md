# Ultimate Smasher
A Super Smash Bros. Ultimate (Nintendo Switch) save editor written in Python

Features
=====
With Ultimate Smasher you can edit the following:

* Gold
* SP
* Hammers
* Tickets
* Snacks



It also allows to:
* Unlock the full roster
* Get All Cores
